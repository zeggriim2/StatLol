Prérequis
-------------
 - PHP >= 7.4
 - Symfony 5.4.*

Commande Initialisation (Division/Queue/Tier)
=======

Si on veut initailisé tout les éléments :
```shell
php bin/console init:init
```

Chaque entité peut-être initialisé indépendamment 
```shell
Exemple :
php bin/console init:init queue
```
